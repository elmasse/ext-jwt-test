# ext-jwt-test

This is a working example explained in the post [Using JWT Tokens with Node & ExtJS 5](http://elmasse.github.io/nodejs/using-jwt-with-extjs.html).

## What you need

- Sencha Cmd 
- Node > 0.10

## Running the example

Install dependencies

```
npm install

```

Start the server

```
node server.js

```
